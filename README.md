# eagle-MangaStream

## 简介

eagle-MangaStream是Eagle软件的一个专用插件，专为漫画阅读体验优化设计。初衷是发现eagle手机漫画图片很方便，加上以前也收集了很多漫画图片，想统一用eagle管理，但是eagle本身没有漫画方式的阅读方式。该插件解决了Eagle原生视图无法实现漫画网站式连续浏览的问题，特别适合已使用Eagle收集和管理的漫画资源。通过本插件，您可以将多张独立漫画图片以**竖向或横向无缝拼接**方式呈现，实现流畅的下拉式/横滑式阅读体验，支持竖屏瀑布流、横屏从左到右、横屏从右到左三种阅读模式，无需手动拼接图片或转换格式。

## 安装方法

### 直接安装

在Eagle中依次选择：插件 → 插件中心 → 搜索MangaStream → 安装

### 源码安装

1. 克隆或下载仓库：`git clone https://github.com/Jiaheliu137/MangaStream`
2. 在Eagle中依次选择：插件 → 开发 → 开发者选项 → 导入本地项目
3. 导入下载的文件夹即可完成安装

## 使用方法

1. 在Eagle资源库中选择需要阅读的漫画图片（支持多选）
2. 点击插件图标启动漫画阅读模式
3. 使用熟悉的浏览器式操作方式浏览内容，快捷键都一样
4. 另外选择新的一些列图片的时候，不用重启插件，直接刷新即可

| 快捷键              | 功能                                                  |
| ------------------- | ----------------------------------------------------- |
| W / S 或 ↑ / ↓      | 页面上下平滑滚动（横屏模式下自动映射为左右滚动）      |
| Space / Shift+Space | 向下/向上翻页                                         |
| M                   | 切换阅读模式（竖屏 → 横屏左到右 → 横屏右到左 → 竖屏） |
| B                   | 切换背景主题色（与Eagle同步，支持6种主题）            |
| Ctrl + 滚轮         | 放大/缩小图片                                         |
| Ctrl + "+"          | 放大视图                                              |
| Ctrl + "-"          | 缩小视图                                              |
| Ctrl + "0"          | 重置缩放比例                                          |
| Ctrl + R            | 刷新图片（更新选中内容后使用）                        |
| Ctrl + E            | 导出当前图片为PDF                                     |
| Ctrl + W            | 关闭阅读窗口                                          |
| Shift+T             | 钉住窗口或者取消钉住窗口                              |
| F                   | 全屏                                                  |
| ESC                 | 退出全屏                                              |
| H                   | 隐藏/显示所有UI按钮（包括模式切换按钮）               |

## 更新内容

### v1.5.1

### v1.5.0

1. **多模式阅读体验**：新增横向画卷阅读模式，支持三种模式自由切换：
   - **竖屏瀑布流**（默认）：经典的从上到下连续阅读
   - **横屏从左到右 (LTR)**：适合日式以外的横向阅读习惯
   - **横屏从右到左 (RTL)**：适合日式漫画的传统阅读方向
   - 点击右下角模式切换按钮或按 `M` 键即可在三种模式间循环切换
2. **模式切换保留阅读进度**：切换阅读模式时自动记住当前页码，切换后直接定位到同一页，无闪烁无跳转。
3. **优雅的模式切换动画**：在切换排版模式时引入 500ms 的淡入淡出遮蔽动画，彻底消除排版重新计算时的视觉闪烁和布局跳动感。
4. **像素级精准坐标映射**：重构了底层的中心点映射数学模型，现在任何缩放比例下的跳转或模式重载，都能确保目标页面的像素绝对中心完美对齐当前屏幕中心，并彻底消除了小图缩放时滚动条越界截断跳回第一页的Bug。
5. **横向模式完整功能适配**：
   - 键盘方向键 W/S/↑/↓ 自动映射为横向滚动
   - 自定义滚动条全面适配横竖模式，支持拖拽操作
   - 放大后的副轴滚动条同样支持拖拽
   - 虚拟滚动引擎完整支持横向布局
6. **UI 改进**：
   - 右下角所有功能按钮的鼠标悬浮文字已全面补充对应的键盘快捷键提示
   - 模式切换按钮受 `H` 键隐藏/显示控制
   - 修复首次打开插件时模式切换按钮不响应的问题
7. **性能优化**：
   - 实现**海量图片模式秒切**：跳过 API 调用与过滤耗时环节，直接复用内存数据，10w+ 张图片切换排版亦能达到 50ms 级别的瞬时无缝衔接。

### v1.4.0

1. **性能体验极限进化 (虚拟滚动重构)**：彻底重写底层图片渲染架构，引入基于精确预计算的虚拟滚动引擎。
   - 现在可以**一瞬间流畅打开20万张以上的漫画图片**，彻底告别海量图片导致的白屏、卡顿与崩溃。
   - 内存占用与 CPU 消耗断崖式下降。网页中始终只渲染屏幕前后的30张图片。
   - 彻底修复大量图片下滚动条跳跃、漂移及自动跑到最后一张的问题。
   - 后台预解码引擎：自动提前解码前方 50 张图片，滑动时无需等待加载。
2. **极速页码跳转**：点击左上角页码指示器，即可输入数字一键精准跳转，O(1) 定位无延迟。
3. **键盘导航与主题切换**：新增 W/S/上下/空格键 翻页功能；新增 T 键沉浸式背景色切换 (纯黑/深灰/纯白)。
4. 修复各种长宽比例图片混排时的滚动误差。
